Node.js, what's it all about?

Node.js is an event driven javascript based framework designed to build highly scalable and efficient network applications. Node functions only run when needed and can support multiple calls at once. This keeps it lightweight and scalable.

Why would you want to run JavaScript on the server side?

JavaScript is perfectly suited for asyncronys tasks. It's also become a ubiquitous programming language understood by most web developers around the world. Node.js was built on the V8 Engine developed by Google (Chrome web browser runs on this) which makes it very fast.

Node Features

  • Real-time communication between the client and server
  • Blocking I/O operations are passed as asynchronys tasks
  • Easy database reading and writing using JSON data serializers
  • HTTP requests are treated like streams which can be processed in real-time
  • NPM, Node Package Manager, for installing publicly available reusable components
    • Express, Sinatra-like web development framework
    • Jade, HAML-like templating engine and the default for the Express framework
    • Coffee-Script, a coffee script compiler written for Node.js applications
    • Socket.io, allows the server to push data to the client through a persistent connection
